# West Hacks Submission Checklist

## Required

- [ ] Record a video demo that is 5 minutes or shorter.
- [ ] Upload the video and copy the share link.
- [ ] Create a public GitHub repository.
- [ ] Push all source code before the deadline.
- [ ] Make sure the repo includes `README.md`, `HACKATHON_SUBMISSION.md`, and `VIDEO_DEMO_SCRIPT.md`.
- [ ] Add the GitHub repo link to Devpost.
- [ ] Add either a working app link or the local run instructions from the README.
- [ ] Fill out the Devpost project description.

## Devpost Answers

Use `HACKATHON_SUBMISSION.md` for:

- What issue are you solving?
- How does your project address it?
- What was the hardest part of the build?

## Before Submitting

- [ ] Confirm `npm install` works.
- [ ] Confirm `npm run dev` opens the app at `http://localhost:3000`.
- [ ] Confirm the check-in form submits and updates the insights.
- [ ] Confirm the video link is public or unlisted but viewable.
- [ ] Confirm the GitHub repo is public.
